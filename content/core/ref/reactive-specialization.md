## Description

You gain an additional reaction per round.

If this is not the first time you have taken this specialization,
the cost changes as per the following table:

| Time | Cost (sp) |
| ---- | --------- |
| 1    | 5         |
| 2    | 10        |
| 3    | 20        |
| 4    | 40        |
